# 簡單的函數

```r
ls() # 目前環境中有哪些變數
rm() # 移除環境中的變數
getwd() # 目前的檔案路徑在哪裡
setwd() # 設定檔案路徑
Sys.Date() # 回傳系統日期
Sys.time() # 回傳系統日期與時間
q() # 離開 RStudio
```
